== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Advanced User Access Manager"
3. **Install** the "Advanced User Access Manager" plugin
4. **Activate** "Advanced User Access Manager" from your Plugins page
